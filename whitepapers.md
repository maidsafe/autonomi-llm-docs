## Technical White Papers
The original technical papers authored by David Irvine, laying out the foundational concepts that underpin the network:

- [Autonomous Network](https://autonomi.com/whitepapers/autonomous-network.md) — David Irvine et al (September 2010) — The core design for a self-managing, serverless network
- [Distributed Hash Table](https://autonomi.com/whitepapers/distributed-hash-table.md) — David Irvine (September 2010) — The DHT implementation enabling decentralised data lookup
- [Self Encrypting Data](https://autonomi.com/whitepapers/self-encrypting-data.md) — David Irvine (September 2010. Revised June 2015) — Convergent encryption with obfuscation for secure storage
- [Self Authentication](https://autonomi.com/whitepapers/self-authentication.md) — David Irvine (September 2010) — Anonymous, serverless user authentication
- [Distributed File System](https://autonomi.com/whitepapers/distributed-file-system.md) — David Irvine (September 2010) — File storage across the decentralised network
